
### 식별자
* 반드시 글자(알파벳 사용 권장), _, $로 시작해야 한다
* 숫자로 시작하는 것은 허용하지 않는다
* 예약어는 식별자로 사용할 수 없다

### 세미콜론
* 문을 구분하는 단위 
* 자바스크립트에서는 현재 문에 다음 문이 이어질 수 없을 때 줄바꿈을 세미콜론 취급한다
* 줄바꿈을 하는 경우에 세미콜론을 생략할 순 있지만, 줄바꿈이 없는 경우엔 반드시 세미콜론으로 문을 구분해야한다.

```tsx
// 세미콜론 생략 가능
a=3
b=4

// 생략 불가능
a=3; b=4;
```
```tsx
let a
a
=
3
console.log(a)

// let a; a=3; console.log(a);
```
